# Thesis App
MVP android restaurant app. 

# Features : 
- closest restaurants, coffees, delivery places list
- reach information about each place
- augumented reality view


# Technologies : 
- RxJava
- RxAndroid
- Dagger 2
- Retrofit
- ButterKnife
- Yel Fusion API
